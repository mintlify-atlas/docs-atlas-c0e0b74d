## Get started in four steps

<Steps>
  <Step title="Install SwitchPalette">
    Download the latest Windows build and run the installer. See [Installation](/installation) for requirements and setup.
  </Step>
  <Step title="Connect your keyboard">
    Plug in your Vial-compatible keyboard over USB. SwitchPalette will automatically detect and connect to it. See [Device Selection](/ui/device-selection).
  </Step>
  <Step title="Remap your keys">
    Open the [Editor Page](/ui/editor-page), click any key on the visual layout, and select a new assignment from the key palette. Changes apply instantly.
  </Step>
  <Step title="Create your first macro">
    Switch to the macro panel in the key palette, choose a macro type (text, combo, AI, and more), and assign it to any key. See [Macros](/features/macros).
  </Step>
</Steps>

<Note>
  SwitchPalette requires a Vial-compatible keyboard firmware. Standard keyboards without Vial firmware will not be detected. See [Device Setup](/configuration/device-setup) for details.
</Note>
