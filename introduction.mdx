SwitchPalette brings together keyboard customization, macro automation, infrared remote management, and AI text processing into a single Windows application. If you use a Vial-compatible custom keyboard, SwitchPalette gives you a visual interface to remap keys, build macros, automate workflows, and control smart home devices — all without leaving your desktop.

## What problems it solves

Managing a custom keyboard usually means juggling separate tools: one for keymaps, another for macros, another for scripting. SwitchPalette consolidates these into one place:

- **Keymap fragmentation** — remap any key on any layer in real time with a visual editor, without flashing firmware
- **Macro limitations** — go beyond simple text snippets to launch apps, send IR signals, or rewrite selected text with Gemini AI
- **IR remote sprawl** — store all your infrared remote codes in Firebase Firestore so they are available across devices
- **Workflow complexity** — chain multiple actions (app launch, notification, macro, wait) into reusable visual workflows

## Technology stack

| Layer | Technology |
|---|---|
| Framework | Flutter |
| Language | Dart (SDK ^3.7.2) |
| State management | Riverpod, flutter_hooks |
| Database / AI | Cloud Firestore (Firebase), Gemini AI (firebase_ai) |
| Desktop integration | bitsdojo_window, tray_manager, windows_notification, win32 |
| Device communication | hid4flutter (HID over USB) |
| Animations | Rive |
| Configuration | shared_preferences, TOML |

SwitchPalette communicates with Vial-compatible keyboards using the raw HID interface via `hid4flutter`. The Vial protocol's `dynamicKeymap` commands (`0x12` get, `0x13` set) are used to read and write keycodes directly over USB without requiring firmware changes.
